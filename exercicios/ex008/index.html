Principais formatações

Negrito

Nesta frase um Termo em negrito usando a tag <b> </b> (não semântica) ou seja so apresenta forma.

Nesta frase temos um termo em destaque usando a tag <strong> </strong> (Semântica) além da forma da sentido também

italico / ênfase

Nesta frase, temos um termo em itálico usando a tag <i> </i> (não semântica).

Nesta frase, temos um termo em itálico usando a tag <em> </em> (semântica).

Texto marcado

Podemos criar tambem um texto marcado usado a tag <mark> </mark>

Texto grande e pequeno

Estamos criando um texto grande com a tag <big> </big>e um texto pequeno com a tag <small> </small> nesse paragrafo.

Texto deletado

Podemos marcar um texto como excluido unsado a tag <del> </del> indicando que pode ser lido mas nao considerada

texto inserido

Podemos marcar um texto como inserido com a tag <ins> </ins> para dar uma enfase que pode ser lido mas nao considerada

Existe tambem o sublinhado com a tag <u> </u> (não semântico)

Texto sobrescrito

para inserir coisas do tipo x20+3 usamos a tag <sup> </sup>

Texto subscrito

para inserir coisas do tipo H2O usamos a tag <sub> </sub>
